Realidad Virtual

Es una simulación con apariencia real, es increíble que podamos recrear un entorno gracias a la tecnología. La inmersión es clave para disfrutar de esta genial herramienta! Usualmente se usan gafas, pero a veces estos dispositivos pueden venir acompañados de cascos, guantes, trajes o incluso mandos!.....


El término realidad virtual (RV) se popularizó a finales de la década de 1980 por Jaron Lanier, uno de los pioneros del campo. Al mismo tiempo, también apareció el término Realidad Artificial. La Enciclopedia Británica describe la realidad virtual como "el uso del modelado y la simulación por computadora que permite a una persona interactuar con un entorno sensorial tridimensional artificial


La realidad virtual comprende dos elementos principales: el entorno del usuario y el entorno virtual. Mientras el usuario interactúa La interfaz puede considerarse como un traductor entre el usuario y el sistema de realidad virtual.
